Data Preprocessing Pipeline

Data Cleaning

The dataset has no missing or corrupt value rows. Every class contains exactly 50 examples, making it a perfectly balanced dataset with no class imbalance issues.

Feature Scaling

We apply StandardScaler to normalize all four numerical features. Since KNN relies heavily on Euclidean distance calculations, unscaled features with wider ranges would disproportionately bias the model.

Train-Test Split

We split the dataset using an 80% training / 20% testing ratio. stratify=y ensures the species ratio is kept equal in both subsets, while a random state of 42 guarantees reproducible results.

Selected Features

We select all 4 geometrical metrics of flower anatomy: Sepal Length, Sepal Width, Petal Length, Petal Width. No dimensionality reduction is performed, as the feature space is low.

Project Details & Algorithm Explanation

Dive deep into the mathematical formulations and theory driving the supervised model.

What is Supervised Learning?

Supervised Learning is a paradigm in Machine Learning where models are trained using labeled datasets. The model analyzes historical training inputs mapped to target outcomes to build a mathematical projection rule. When presented with unseen data, it calculates predictions matching the learned map.

What is Classification?

Classification is a supervised machine learning task targeting categorical labels. The goal is to isolate target categories (e.g., specific flower species) based on geometric features. Unlike regression, output formats are discrete variables.

What is K-Nearest Neighbors (KNN)?

KNN is a non-parametric, lazy supervised learning algorithm. It does not train explicit parameters or fit weights. Instead, it memorizes all training samples. When classifying a new data point, it looks at the nearest K historical points in the scaled vector space and determines the output class using a majority vote.

Euclidean Distance Formulation

We calculate similarity using the Euclidean distance metric. For two coordinates $P$ and $Q$ across $n$ features:

Distance = √ ∑ (p_i - q_i)²
Scaling coordinates using a standardizer ensures features with wider ranges do not dominate the calculation.

Algorithm Analysis

Advantages
  • Intuitively simple to understand and deploy.
  • Lazy learner: Requires no training phase, making updates to the training database instant.
  • Versatile for multi-class classification problem spaces.
Disadvantages
  • High computational complexity during prediction: Must measure distances to every training record.
  • Sensitive to noisy data, outliers, and irrelevant background features.
  • Susceptible to memory boundaries: Becomes extremely slow as dataset volumes grow.
